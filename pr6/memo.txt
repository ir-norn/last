シーン間のデータやり取りは ARGVを使う

問題点として、１フレームに複数のノードを作る場合にARGVが埋まってる
ノード ＝ ファイル　
なので

ARGV.extend Module.new { attr_accessor :Remilia }
ARGV.Remilia.extend　＜＜ method_missing
ARGV.Remilia.FILENAME = Hash

１フレームに同じファイル名のノードを複数作る場合の
引数フォーマットはとりあえず
{ :x => 10 , :hash => { :x => 20 , :hash => { :x => 30 } } }

再帰で取り出して読み捨てていく
HashじゃなくてArrayでも良い
